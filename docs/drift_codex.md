# THE FIRST NIGHT (DRIFT) — Codex
*The living catalogue (Charter §I.16). One row per entry; the four gates abbreviated as Res / Tools / Know / Skill / Time. **Experience role** (Charter §I.18, D-016) names the feeling each entry exists to create — content with no experience role is cut. Status = the cycle that shipped it, or P1/P2 = planned for that phase. Whichever cycle introduces an entry appends it and fills its gates precisely. Canonical home: repo `/docs/` from Cycle 01 onward.*
